Path Length Exceeded

We may be unable to create the folders needed to save these image files. If you are on a Windows system, this could mean the path length needed to save the images in a nested folder structure of collection > study > series > patient is too long for your file system. Windows has a maximum file path length of 260 characters, and these files will need around characters, not including the path to your chosen destination folder (which may bring the total path length over the maximum allowed by Windows).

We can save your files in a flat folder structure under only the collection ID, and provide a manifest which maps the name of each file to its collection, patient, study, and series. Alternatively, you can cancel and choose a different destination folder.

If you are not on a Windows system and are still seeing this error message, please double-check that you have sufficient disk space and are allowing the browser to save the files when prompted for access.
